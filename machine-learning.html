About me

My name is Safiq Faray. I'm a Web Developer and Machine Learning Enthusiast specializing in full-stack development and data-driven solutions. I have expertise in JavaScript (Node.js, React, Next.js, NestJS) and PHP (Laravel), along with a strong foundation in Python and Java for algorithmic problem-solving and machine learning applications. With a Bachelor's degree in Informatics from Institut Teknologi Bandung, I integrate my academic knowledge with practical experience in machine learning, deep learning, and data analytics. My experience includes data preprocessing, feature engineering, ML algorithms (logistic regression, SVM, linear regression), and deep learning models for NLP (transformers) and image processing. I am also proficient in data visualization using Looker Studio, ensuring insightful analysis for decision-making. Driven by continuous learning, I strive to develop efficient, scalable, and intelligent systems that bridge the gap between software engineering and AI-driven solutions.
